# 🧪 Расчет чистого спирта в AlcoTracker

## Обзор

Начиная с версии с поддержкой расчета чистого спирта, AlcoTracker учитывает **крепость напитков** при подсчете статистики. Это позволяет более объективно оценивать потребление алкоголя.

## Почему это важно?

**500 мл водки ≠ 500 мл пива**

- 500 мл водки (40%) = **200 мл чистого спирта**
- 500 мл пива (5%) = **25 мл чистого спирта**

Разница в **8 раз**! Поэтому важно считать не только объем напитков, но и количество чистого алкоголя (этанола).

---

## 📊 Метрики

Теперь система показывает **две метрики**:

### 1. **Объем напитков** (как раньше)
- Сколько миллилитров напитка вы выпили
- Полезно для понимания количества жидкости

### 2. **Чистый спирт** (новое)
- Сколько миллилитров чистого этанола вы употребили
- **Более объективная метрика** для оценки влияния на здоровье
- Учитывает крепость каждого напитка

---

## 🧮 Формула расчета

```
Чистый спирт (мл) = Объем напитка (мл) × Крепость (%) / 100
```

### Примеры:

1. **Бутылка пива** (500 мл, 5%)
   ```
   500 × 5 / 100 = 25 мл чистого спирта
   ```

2. **Бокал вина** (200 мл, 12%)
   ```
   200 × 12 / 100 = 24 мл чистого спирта
   ```

3. **Рюмка водки** (50 мл, 40%)
   ```
   50 × 40 / 100 = 20 мл чистого спирта
   ```

## 📈 Графики статистики

### График 1: Динамика потребления (объем)
- Показывает объем выпитых напитков по дням
- Линия тренда с направлением (рост/снижение)
- Скользящее среднее для сглаживания

### График 2: По типам напитков (объем)
- Сравнение объемов разных напитков
- Столбчатая диаграмма

### График 3: Динамика чистого спирта ⭐
- **Самый важный график**
- Показывает реальное количество алкоголя
- Учитывает крепость напитков
- Линия тренда + скользящее среднее
- Дополнительная аналитика:
  - Общее количество чистого спирта
  - **Максимальная серия дней употребления подряд** (негативная метрика)
  - **Максимальная серия дней БЕЗ употребления подряд** ✅ (позитивная метрика)
  - Максимум за один день

## 📊 Streak Tracking (Серии дней)

Система автоматически отслеживает паттерны употребления:

### Максимальная серия дней С употреблением подряд ⚠️
- Показывает самую длинную непрерывную серию дней, когда был алкоголь
- **Негативная метрика** - чем меньше, тем лучше
- Пример: если вы пили каждый день с пятницы по воскресенье = 3 дня подряд

### Максимальная серия дней БЕЗ употребления подряд ✅
- Показывает самую длинную непрерывную серию трезвых дней
- **Позитивная метрика** - чем больше, тем лучше
- Выделена зеленым цветом для мотивации
- Пример: если вы не пили с понедельника по четверг = 4 дня подряд

### Интерпретация:

```
Пример 1: Умеренное употребление
- Макс. дней подряд: 2 дня
- Макс. дней без спирта: 12 дней
→ Здоровый паттерн: короткие периоды употребления, длинные периоды трезвости

Пример 2: Частое употребление
- Макс. дней подряд: 14 дней
- Макс. дней без спирта: 2 дня
→ Тревожный паттерн: длинные периоды употребления, короткие периоды трезвости

Пример 3: Выходные
- Макс. дней подряд: 2 дня (Пт-Сб или Сб-Вс)
- Макс. дней без спирта: 5 дней (Пн-Пт)
→ Классический паттерн "употребление по выходным"
```

### Цель:
- Стремитесь увеличить "Макс. дней без спирта" ✅
- Стремитесь уменьшить "Макс. дней подряд" ⚠️

## 🎯 Практические примеры

### Пример 1: Вечер с пивом
```
Выпито: 3 бутылки пива по 500 мл
Объем: 1500 мл
Чистый спирт: 1500 × 5% = 75 мл
```

### Пример 2: Вечер с водкой
```
Выпито: 3 рюмки водки по 50 мл
Объем: 150 мл
Чистый спирт: 150 × 40% = 60 мл
```

**Вывод:** 3 бутылки пива ≈ 3 рюмки водки по чистому спирту!

### Пример 3: Смешанный вечер
```
- 1 бутылка пива (500 мл, 5%) = 25 мл спирта
- 1 бокал вина (200 мл, 12%) = 24 мл спирта
- 2 рюмки водки (100 мл, 40%) = 40 мл спирта

Итого:
- Объем: 800 мл напитков
- Чистый спирт: 89 мл (это важная цифра!)
```

## 💡 Рекомендации ВОЗ

Всемирная организация здравоохранения рекомендует:

- **Мужчины:** не более 40 мл чистого спирта в день
- **Женщины:** не более 20 мл чистого спирта в день
- **Безопасная норма в неделю:** не более 210 мл (мужчины) / 140 мл (женщины)

### Пересчет в напитки:

**40 мл чистого спирта = это:**
- 800 мл пива (1.6 бутылки)
- 320 мл вина (1.5 бокала)
- 100 мл водки (2 рюмки)

## 🔧 Техническая реализация

### Backend (PHP/Laravel)

1. **AlcoholType Enum** - хранит крепость каждого типа напитка
   ```php
   public function alcoholPercentage(): float
   {
       return match($this) {
           self::BEER => 5.0,
           self::VODKA => 40.0,
           // ...
       };
   }
   ```

2. **AlcoholEntry Model** - расчет чистого спирта
   ```php
   public function getPureAlcoholMlAttribute(): float
   {
       return $this->alcohol_type->calculatePureAlcohol($this->amount_ml);
   }
   ```

3. **AlcoholEntryService** - агрегация статистики
   - Группировка по дням
   - Суммирование чистого спирта
   - Расчет трендов

### Frontend (Vue/Nuxt)

1. **TrendChart.vue** - новый компонент
   - Линейный график с трендовой линией
   - Скользящее среднее (7 дней)
   - Линейная регрессия для тренда

2. **Statistics Page** - обновленная страница
   - 4 графика вместо 2
   - Разделение на объем и чистый спирт
   - Дополнительная аналитика

## ❓ FAQ

**Q: Почему крепость фиксированная?**
A: Мы используем средние значения для каждого типа напитка. В будущем можно добавить возможность указывать точную крепость.

**Q: Можно ли изменить крепость для своих записей?**
A: Пока нет, но это в планах на будущее.

**Q: Откуда взяты значения крепости?**
A: Из стандартных значений для каждого типа напитка + данные производителей.

**Q: Почему граммы отличаются от миллилитров?**
A: Плотность этанола = 0.789 г/мл, поэтому 100 мл спирта = 78.9 г.
