# AlcoTracker

Веб-приложение для личного учёта потребления алкоголя с аналитикой. Пользователь фиксирует напитки, просматривает историю и получает наглядную статистику — от дневной сводки до графиков трендов и расчёта чистого спирта.

> AlcoTracker — инструмент самонаблюдения. Он не ставит диагнозов и не заменяет медицинскую консультацию.

## Аутентификация

### Регистрация

- Имя, email, пароль и подтверждение пароля
- После успешной регистрации — автоматический вход и переход на главную

### Вход

- Email и пароль
- Опция «Запомнить меня»
- Ссылка на восстановление пароля
- Ссылка на регистрацию
- После входа — переход на страницу из параметра `redirect` или на главную

### Выход

- Кнопка «Выйти» в шапке защищённых страниц
- Завершение сессии и переход на страницу входа

### Восстановление пароля

1. **Забыли пароль** — ввод email, отправка письма со ссылкой для сброса
2. **Сброс пароля** — установка нового пароля по ссылке из письма (token + email в URL)

Неавторизованный пользователь при попытке открыть защищённую страницу перенаправляется на `/login` с сохранением исходного адреса в `redirect`.

## Главная страница

Центральный экран для ежедневного учёта.

### Приветствие

- Персональное приветствие с именем пользователя
- Краткое описание назначения сервиса

### Выбор даты

- Календарь с ограничением: нельзя выбрать будущую дату
- Кнопки «предыдущий день», «следующий день», «Сегодня»
- Все записи и список ниже привязаны к выбранной дате

### Добавление записи

| Поле | Обязательное | Описание |
|------|:------------:|----------|
| Тип алкоголя | да | Выбор из 13 типов напитков |
| Количество (мл) | да | От 1 до 5000 мл |
| Дата | да | Синхронизируется с выбранной датой, не в будущем |
| Комментарий | нет | До 500 символов, со счётчиком |

После добавления форма очищается (кроме даты), список записей и недельная статистика обновляются автоматически.

### Список записей за день

- Заголовок с относительной датой: «сегодня», «вчера» или полная дата
- Суммарный объём за выбранный день
- Карточки записей: тип напитка, объём, комментарий, время добавления
- Удаление записи с подтверждением

### Краткая статистика за неделю

Блок на главной с данными за последние 7 дней:

- **Всего выпито** — суммарный объём в мл
- **Количество записей**
- **Разбивка по типам** — сколько мл каждого напитка

## Статистика

Отдельная страница с развёрнутой аналитикой за выбранный период.

### Фильтр периода

| Период | Описание |
|--------|----------|
| За неделю | Последние 7 дней |
| За месяц | С 1-го числа текущего месяца |
| За 3 месяца | С 1-го числа трёх месяцев назад |
| За полгода | С 1-го числа шести месяцев назад |
| За год | С 1-го числа двенадцати месяцев назад |
| С начала года | С 1 января текущего года |

По умолчанию выбран «За месяц». Под фильтром отображается точный диапазон дат.

### Динамика потребления и тренд

Линейный график с тремя линиями:

- фактическое потребление по дням
- скользящее среднее (7 дней)
- линия тренда (линейная регрессия)

Дополнительные показатели: общий объём, среднее в день, число дней с записями, максимум за один день.

### Потребление по количеству

Столбчатый график — объём выпитого по дням за период.

### Потребление по типам напитков

Столбчатый график — распределение объёма между видами напитков.

### Статистика по чистому спирту

Расчёт с учётом крепости каждого типа напитка. Например: 500 мл водки (40%) = 200 мл чистого спирта, 500 мл пива (5%) = 25 мл.

- График динамики чистого спирта по дням
- Общий объём чистого спирта за период
- Среднее в день
- Максимальная серия дней **с** употреблением подряд
- Максимальная серия дней **без** употребления подряд
- Максимум чистого спирта за один день

### Анализ по дням недели

Круговая диаграмма распределения чистого спирта по дням недели (Пн–Вс).

Автоматические инсайты:

- **Самый активный день** — день с наибольшим потреблением
- **Самый спокойный день** — день с наименьшим потреблением
- **Паттерн** — преобладание выходных, будних дней или равномерное распределение

При отсутствии данных за период каждый блок показывает понятное пустое состояние.

## Типы напитков

В системе 13 типов алкоголя с фиксированной крепостью для расчёта чистого спирта:

| Напиток | Крепость |
|---------|:--------:|
| Пиво | 5.0% |
| Сидр | 5.0% |
| Вино сухое | 12.5% |
| Вино полусладкое | 11.0% |
| Шампанское | 12.0% |
| Ликёр | 20.0% |
| Водка | 40.0% |
| Виски | 40.0% |
| Ром | 40.0% |
| Джин | 40.0% |
| Текила | 40.0% |
| Чача | 50.0% |
| Самогон | 45.0% |

**Формула:** чистый спирт (мл) = объём напитка (мл) × (крепость / 100)

## Технологии
